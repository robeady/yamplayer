Designing the player widget

## Use cases

primary controls:

-   pause / unpause
-   skip next, restart / skip previous
-   change volume / mute
-   see time in song, seek

secondary controls:

-   toggle shuffle
-   toggle repeat
-   stop at end of song

song reactions:

-   see what's playing
-   see/set/change rating
-   add to library
-   remove from library

the queue:

-   see what's next and what's coming
-   see what's passed (and react to it e.g. add to library)

other:

-   show lyrics
